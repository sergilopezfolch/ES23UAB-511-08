# Projecte QueMenges - Enginyeria del Software 2025-26

## Descripció del Projecte
QueMenges és una plataforma innovadora que permet als usuaris accedir a menús casolans preparats per persones individuals. L'aplicació connecta cuiners particulars amb clients, oferint un servei de comanda i repartiment de menjar alternatiu a les grans cadenes comercials.
